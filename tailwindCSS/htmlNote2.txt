Marker Types
css list-style-type property can be set to change markers to circles, squares, or other symbols.
e.g
li{
list-style-type:circle,square,decimal
}
U can remove default padding from list. Do below:
ul{
padding:0; //Removes default padding
}
A hyperlink is specified using HTML tag <a>.This tag is called the anchor tag and anything bw the opening <a>tag
and the closing </a> tag becomes a link and the user can click on it to reach the linked document.
