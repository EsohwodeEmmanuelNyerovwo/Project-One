Find the min/max element of an Array using JavaScript

Finding the minimum or maximum element of an array is a common task in JavaScript, especially in data analysis, optimization problems, and algorithmic comparisons. JavaScript offers several methods to achieve this, each with its advantages. In this article, well explore various approaches to finding the smallest and largest values in an array.

Below are the methods to find the minimum and maximum elements in an array using JavaScript:

Table of Content

  • Using Math.min() and Math.max() Methods 
  • Iterating through the Array 
  • Using a Custom Comparator with Array.sort() Method

Using Math.min() and Math.max() Methods 

The Math objects Math.min() and Math.max() methods are static methods that return the minimum and maximum elements of a given array. The spread() operator could pass these functions into an array. The spread operator allows an iterable to expand in places where multiple arguments are expected. In this case, it automatically expands the array and gives the numbers to the functions. 

Syntax:

minValue = Math.min(...array);
maxValue = Math.max(...array);

Example: We will see the basic implementation of JavaScript Math.min() and Math.max() methods.

JavaScript 
let Arr = [50, 60, 20, 10, 40];
let minValue = Math.min(...Arr);
 let maxValue = Math.max(...Arr);
console.log("Minimum element is:" + minValue);
console.log("Maximum Element is:" + maxValue);

Iterating through the Array 

Iterating through the array and keeping track of the minimum and maximum elements. The minimum and maximum element can be kept track by iterating through all the elements in the array and updating the minimum and maximum element up to that point by comparing them to the current minimum and maximum values. The minimum and maximum values are initialized to Infinity and -Infinity. 

Example: We will be iterating through the elements of the array to find the max and min elements from the array.

JavaScript 
let Arr = [50, 60, 20, 10, 40];
let minValue = Infinity;
let maxValue = -Infinity;

 for (let item of Arr) {

        // Find minimum value
        if (item < minValue)
            minValue = item;

        // Find maximum value
        if (item > maxValue)
            maxValue = item;
console.log("Minimum element is:" + minValue);
console.log("Minimum element is:" + maxValue);
}

Using a Custom Comparator with Array.sort() Method

The Array.sort() method sorts the elements of an array in place and returns the sorted array. By providing a custom comparator function to Array.sort(), you can sort the array in ascending or descending order and then easily access the minimum and maximum elements.

Example: In this example, we will sort the array in ascending order to get the minimum element and in descending order to get the maximum element.

JavaScript 
let Arr = [50, 60, 20, 10, 40];
// Sort array in ascending order
Arr.sort((a, b) => a - b);
let minValue = Arr[0]; // First element after sorting is the minimum
 // Sort array in descending order
 Arr.sort((a, b) => b - a);
let maxValue = Arr[0]; // First element after sorting is the maximum
console.log("Minimum element is:" + minValue);
console.log("Maximum Element is:" + maxValue);
